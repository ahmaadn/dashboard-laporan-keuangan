
DOKUMENTASI KEBUTUHAN SISTEM

Nama Aplikasi

Aplikasi Pengelolaan Keuangan UMKM Kerajinan Kulit Berbasis Web dengan Fitur Dashboard Interaktif

Deskripsi Singkat

Aplikasi digunakan untuk membantu UMKM dalam mengelola data keuangan berupa pemasukan dan pengeluaran secara terstruktur. Selain itu, aplikasi menyediakan Dashboard Interaktif yang menampilkan ringkasan kondisi keuangan usaha sehingga pemilik dapat memantau kondisi usaha dengan lebih mudah.

---

FITUR APLIKASI

1. Login
2. Kelola Data Produk
3. Kelola Data Pemasukan
4. Kelola Data Pengeluaran
5. Kelola Data Pengguna
6. Laporan Keuangan
7. Dashboard Interaktif

HAK AKSES ADMIN

1. Mengelola Data Produk
2. Mengelola Data Pemasukan
3. Mengelola Data Pengeluaran
4. Mengelola Data Pengguna
5. Melihat Dashboard
6. Melihat Laporan

HAK AKSES PEGAWAI

1. Mengelola Data Pemasukan
2. Mengelola Data Pengeluaran
3. Melihat Data Produk
4. Melihat Dashboard (jika diberikan hak akses)

FITUR DASHBOARD INTERAKTIF

1. Ringkasan Keuangan
   
   - Total Pemasukan
   - Total Pengeluaran
   - Laba/Rugi
   
   Interaktif:
   
   - Klik Total Pemasukan → menampilkan detail seluruh data pemasukan sesuai periode yang dipilih.
   - Klik Total Pengeluaran → menampilkan detail seluruh data pengeluaran.
   - Klik Laba/Rugi → menampilkan rincian perhitungan laba/rugi.

2. Grafik Tren Pemasukan dan Pengeluaran
   
   Interaktif:
   
   - Klik salah satu batang atau titik grafik pada bulan tertentu → menampilkan rincian transaksi pada periode tersebut.
   - Grafik akan berubah otomatis ketika pengguna mengubah periode.

3. Analisis Pengeluaran per Kategori
   
   Menampilkan:
   
   - Bahan Baku
   - Operasional
   - Pengiriman
   
   Interaktif:
   
   - Klik salah satu kategori pada diagram → menampilkan rincian pengeluaran berdasarkan kategori yang dipilih.

4. Tren Penjualan Produk
   
   Menampilkan perkembangan penjualan produk dari waktu ke waktu.
   
   Interaktif:
   
   - Klik nama produk → menampilkan detail riwayat penjualan produk tersebut.
   - Grafik akan berubah sesuai periode yang dipilih.

5. Produk Terlaris
   
   Menampilkan produk dengan jumlah penjualan tertinggi.
   
   Interaktif:
   
   - Klik produk → menampilkan informasi jumlah penjualan dan riwayat transaksi produk tersebut.

6. Daftar Transaksi Terkini
   
   Menampilkan transaksi pemasukan dan pengeluaran terbaru.
   
   Interaktif:
   
   - Klik transaksi → menampilkan detail transaksi secara lengkap.

7. Filter Periode Interaktif
   
   Menyediakan filter:
   
   - Hari Ini
   - Minggu Ini
   - Bulan Ini
   - Tahun Ini
   - Rentang Tanggal Tertentu
   
   Interaktif:
   
   - Saat pengguna mengubah periode, seluruh komponen dashboard seperti ringkasan keuangan, grafik, produk terlaris, dan transaksi terkini akan diperbarui secara otomatis sesuai periode yang dipilih tanpa perlu memuat ulang halaman.
8. Perbandingan Periode (Period Comparison)

   Menampilkan perbandingan total pemasukan, pengeluaran, dan laba/rugi antara dua periode yang dipilih secara berdampingan, dilengkapi dengan persentase kenaikan atau penurunan.

   Interaktif:

   - Pengguna dapat memilih dua periode pembanding secara bebas, misalnya bulan ini dengan bulan lalu, atau bulan ini dengan periode yang sama tahun sebelumnya.
   - Klik salah satu nilai pada hasil perbandingan → menampilkan rincian transaksi pada periode terkait.
